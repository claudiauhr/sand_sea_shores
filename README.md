# Sand Sea Shores

## Description

A web-based application to create a post about sea and shore places and attractions. You can organize your vacations by choosing places, foods, and places to relax or have fun in advance.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Python
- Django
- Git/GitHub
- PostgreSQL
- Heroku
